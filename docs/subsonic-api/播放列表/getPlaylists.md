# getPlaylists

返回所有播放列表。

**参数：**
无

**响应：**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<subsonic-response xmlns="http://subsonic.org/restapi" status="ok" version="1.16.1">
    <playlists>
        <playlist id="123" name="我的播放列表" owner="用户名" public="false" songCount="10"/>
    </playlists>
</subsonic-response>
```
